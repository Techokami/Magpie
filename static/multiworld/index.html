• Generate your seed (online generator - change player_count=2 to match the number of players you want)
  • Download the BizHawk emulator
  • Place socket.dll in the dll folder inside BizHawk's folder (e.g. BizHawk-2.8-win-x64\dll\socket.dll)
  • Start BizHawk (EmuHawk.exe) and load the seed you generated
  • Go to Tools->Lua Console
  • Open bizhawkConnector.lua
  • If everything is working, the Lua console and LADX should look like this
  • If it doesn't work, the multiworld server may be down - you can try an alternate server by using a different connector script
